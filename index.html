Summary: Weather - Latitude

The purpose of this research was to analyse how weather changes as you get closer to the equator. To accomplish this, first the data was pulled from the OpenWeatherMap API to assemble a dataset on over 500 cities

After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs latitutde. Factors that were analysed included: temperature, cloudiness, wind speed, and humidty. This site provides the source data and visualisations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
